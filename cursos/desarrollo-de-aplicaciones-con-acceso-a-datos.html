Persistencia y Eficiencia

Un software sin persistencia es un software sin memoria. En este curso, aprenderás a gestionar la información de manera profesional utilizando el ecosistema Python.

Iniciamos con la manipulación de flujos de datos en archivos planos, escalamos a la robustez del modelo relacional con PostgreSQL, y finalizamos con la agilidad de los servicios web y la flexibilidad de NoSQL.

Modelado Relacional

Entenderemos cómo estructurar el mundo real en tablas optimizadas y normalizadas.

Cloud & APIs

Transformaremos datos estáticos en recursos dinámicos accesibles mediante microservicios modernos.

Ruta de Exploración

Bloque 1 Persistencia Plana y Bases
Archivos de Texto e IDE
  • Taxonomía: Clasificación de estructuras de datos y persistencia.
  • Puesta en marcha: Configuración de IDE (VS Code/PyCharm) para gestión de datos.
  • Mantenimiento: CRUD profesional sobre archivos de texto planos.
Bloque 2 Mundo Relacional (SQL)
PostgreSQL y Patrón MVC
  • Servidores DB: Instalación y puesta en marcha de un motor relacional.
  • DB-API: Conexión nativa desde Python hacia bases de datos.
  • CRUD DB: Desarrollo de aplicaciones conectadas a DB relacionales.
  • Modelo MVC: Descripción y responsabilidades de las capas en acceso a datos.
Bloque 3 Abstracción y Servicios
ORM, NoSQL y APIs
  • ORM (SQLAlchemy): Puesta en marcha y CRUD usando mapeo objeto-relacional.
  • NoSQL: Definición y uso de bases de datos documentales (MongoDB).
  • Web Services: Implementación de servicios REST para acceso a datos.
