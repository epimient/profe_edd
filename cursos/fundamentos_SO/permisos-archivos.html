Seguridad Industrial: Permisos de archivos

En Linux, tú decides quién puede ver, tocar o ejecutar tus archivos. El poder es tuyo.

Lectura, Escritura y Ejecución

Cada archivo tiene 3 tipos de permisos asignados a 3 entidades diferentes.

Las Entidades

  • User (u)
    El dueño del archivo.
  • Group (g)
    El grupo de usuarios del dueño.
  • Others (o)
    El resto del mundo.

Los Permisos

r

Read (Lectura)

Valor octal: 4

Ver contenido del archivo o listar directorio.

w

Write (Escritura)

Valor octal: 2

Modificar, borrar o renombrar.

x

Execute (Ejecución)

Valor octal: 1

Correr como programa (script).

Comando chmod (Change Mode)

Podemos cambiar permisos sumando los valores octales.

Ejemplo: Permiso total (7) al dueño, lectura y ejecución (5) al grupo, y nada (0) a otros.

chmod 750 archivo_secreto.txt
  • 7 = 4 (r) + 2 (w) + 1 (x)
  • 5 = 4 (r) + 0 + 1 (x)
  • 0 = 0 + 0 + 0

Comando chown (Change Owner)

Cambia el dueño de un archivo.

sudo chown usuario:grupo archivo.txt
