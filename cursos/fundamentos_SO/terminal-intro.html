Primeros pasos con la Terminal

Deja de tenerle miedo a la pantalla negra. Aquí es donde ocurre la verdadera magia.

¿Por qué usar la terminal?

La Interfaz de Línea de Comandos (CLI) es más rápida, potente y consume menos recursos que la Interfaz Gráfica (GUI). Es la herramienta fundamental de cualquier administrador de sistemas.

Comandos Básicos de Supervivencia

1. ¿Dónde estoy? (Print Working Directory)

Muestra la ruta completa del directorio actual.

pwd
2. Listar archivos (List)

Muestra qué hay en el directorio.

ls
ls -l  # Lista detallada (permisos, tamaño, dueño)
ls -a  # Muestra archivos ocultos
3. Cambiar de directorio (Change Directory)

Para navegar por el árbol de directorios.

cd Documentos   # Entra a Documentos
cd ..           # Sube un nivel
cd ~            # Vuelve a tu carpeta personal (Home)

Tip Pro

No tienes que escribir los nombres completos. Escribe las primeras letras y presiona la tecla TAB para autocompletar.

Ejercicio Práctico

  1. Abre tu terminal.
  2. Escribe mkdir curso_linux para crear una carpeta.
  3. Entra en ella con cd curso_linux.
  4. Crea un archivo vacío con touch prueba.txt.
  5. Verifica que está ahí con ls -l.
