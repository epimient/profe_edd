El Kernel: El director de orquesta

El componente central que tiene el control total sobre todo lo que ocurre en el sistema.

¿Qué es el Kernel?

El kernel (o núcleo) es la parte fundamental del sistema operativo. Es el primer programa que se carga en la memoria al arrancar (después del bootloader) y permanece allí hasta que se apaga el equipo.

Nota: El usuario NUNCA interactúa directamente con el kernel. Usamos la "shell" o aplicaciones para enviarle peticiones (System Calls).

Responsabilidades Clave

Gestión de Memoria RAM

Controla cuánta memoria usa cada programa y qué partes de la memoria están libres.
Gestión de Procesador (CPU)

Decide qué aplicación usa el procesador en cada milisegundo (Scheduling).

Controladores de Dispositivos (Drivers)

Actúa como traductor entre el software y el hardware (tarjeta gráfica, disco, teclado).

Modes de Ejecución

Para proteger el sistema, los CPUs modernos operan en dos modos:

  1. Modo Kernel (Ring 0): Acceso total al hardware. Aquí vive el Kernel. Si algo falla aquí, el sistema completo se cae (Kernel Panic / Pantallazo Azul).
  2. Modo Usuario (Ring 3): Acceso restringido. Aquí viven tus aplicaciones (Navegador, Word, Juegos). Si algo falla aquí, solo se cierra el programa.
