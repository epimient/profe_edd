Flet

Apps modernas con el poder de Flutter.

¿Qué es Flet?

Flet es un framework que permite construir aplicaciones web, móviles y de escritorio en Python usando los componentes visuales de Flutter sin necesidad de aprender Dart.

import flet as ft

def main(page: ft.Page):
    page.add(ft.Text(value="¡Hola, Flet!"))

ft.app(target=main)

El Futuro de las GUIs en Python

Flet ofrece una experience multi-dispositivo real, permitiendo que el mismo código se ejecute en el navegador, en Android/iOS o como una app local.
